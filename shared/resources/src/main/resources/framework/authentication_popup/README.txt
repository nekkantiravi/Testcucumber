How to build exe file from .rb
===============================
  $ ocra --console --dll AutoItX3.dll <ruby file>.rb

Prerequisite
============
  - install AutoItX3: https://www.autoitscript.com/site/autoit/downloads/
  - install gems: 'au3', 'ocra' gem
  - copy AutoItX3.dll (32bit) to C:\Ruby200\bin before run 'ocra'
